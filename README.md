# saima-test-task-BE

It's a simple Node.JS app which helps to download videos either from youtube and coursera
It's important to consider that there is no packages for JS to download coursera videos

Here is the guide for running the app

## Environment

Create `.env` file in the root directory and move variables from `.env.example` file
## Dependencies

Run `yarn install`

## Development server

Run `yarn start` to run the server. the app will be run on http://localhost:8000/. Also the app won't automatically reload if you change any of the source files, fill free to use nodemon or other packages.
